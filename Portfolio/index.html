ResortEase – Online Booking System

Description:
ResortEase is a full-featured MERN stack project designed for a resort's online room booking and management system. It includes secure user and admin login, dynamic room listing, booking form with coupon-based discounts, QR code-based UPI payment, and an admin dashboard for managing bookings and availability. We developed ResortEase for Royal Castle Farm Stay Resort, located in Odathurai, Perundurai. This project was created as part of a consultancy effort to digitalize the resort's operations. The system provides seamless user and admin experiences, from room browsing to QR code-based UPI payments. We are grateful to Ms. Parameshwari, CEO of the Resort, who placed her trust in us and gave us this opportunity to transform their booking process through technology.

Technologies Used:
> React.js for Frontend
> Node.js & Express.js for Backend
> MongoDB for Database
